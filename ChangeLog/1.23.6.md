# 1.23.6 (2026-01-08)

## Features
- Added `-output` flag supporting `json` and `text` (default) formats.
- Implemented JSON output for `getevents`, `getinbox`, `getschedule`, `exportinbox`, and `searchandexport` actions.
- Moved all application logging to `stderr` to ensure `stdout` contains only requested data (like JSON) for easier piping.

## Improvements
- Enhanced configuration validation for output formats.
