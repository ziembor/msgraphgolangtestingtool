## [1.14.2] - 2026-01-03

### Added
- **New `-bodyHTML` flag** for HTML email body support
  - Enables sending HTML-formatted emails via `-action sendmail`
  - When provided, email is sent as HTML instead of plain text
  - Supports environment variable `MSGRAPHBODYHTML`
- **New `-attachments` flag** for file attachments
  - Comma-separated list of file paths to attach
  - Supports any file type with automatic MIME type detection
  - Supports environment variable `MSGRAPHATTACHMENTS`
  - Files are base64-encoded and sent via Graph API
- **Multipart MIME support** for `sendmail` action
  - Text body with `-body` flag (default)
  - HTML body with `-bodyHTML` flag
  - Attachments with `-attachments` flag
  - All can be combined for rich email content

### Changed
- Updated CSV logging for `sendmail` to include Body Type and Attachment count
- Enhanced verbose output to show body type (Text/HTML) and attachment details
- Total environment variable count increased from 18 to 20 parameters
