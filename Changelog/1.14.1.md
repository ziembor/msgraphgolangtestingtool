## [1.14.1] - 2026-01-03

### Changed
- **VERSION file is now single source of truth** for versioning
  - Go source code now uses `//go:embed VERSION` to read version at compile time
  - No longer need to manually update version constant in source code
  - Added `embed` package import to `src/msgraphgolangtestingtool.go`
  - Replaced `const version = "x.x.x"` with embedded version variable
- **Changelog reorganization** - split monolithic CHANGELOG.md into individual files
  - Created `Changelog/` folder for version-specific changelog files
  - Each version has its own file: `Changelog/{version}.md`
  - Easier to manage and track changes per release
- **Updated version management documentation** (`.claude/version-management.md`)
  - Simplified procedures: only need to update `src/VERSION` and create changelog file
  - Removed references to updating source code constant
  - Added changelog folder structure documentation
